# はじめに

Claude Codeで作成したツールのドキュメンテーション管理リポジトリへようこそ！このガイドでは、カスタムコマンドを活用した効率的なドキュメント生成方法について説明します。

## 🎯 このリポジトリの目的

Claude Codeで作成したツールのドキュメンテーションを**効率的に生成・管理**するためのシステムです。

### 主な用途
- **Claude Codeで作成したツール**のドキュメント生成
- **カスタムコマンド**による自動ドキュメント生成
- **統一された品質**でのツール紹介
- **多言語対応**による情報共有

## 🚀 基本的な使い方

### ステップ1: 環境準備

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run docs:dev
```

ドキュメンテーションサイトは `http://localhost:5173` で利用可能になります。

### ステップ2: ツールのドキュメント生成

作成したツールのドキュメントを生成する際は、以下のカスタムコマンドを使用できます：

#### 📋 **利用可能なコマンド**

1. **`doc-full [project-path|project-name]`** - 完全なドキュメント生成
   - 概要、インストール、使用方法、API、トラブルシューティングを一括生成
   - 最も包括的なドキュメント作成

2. **`doc-api [project-path|project-name]`** - APIドキュメント生成
   - 関数、インターフェース、型定義を解析してAPIリファレンス生成

3. **`doc-update [project-path|project-name]`** - ドキュメント更新
   - 既存ドキュメントをコードベースの変更に合わせて更新

4. **`translate-docs [project-name]`** - 日本語翻訳
   - 英語ドキュメントを日本語に翻訳

5. **`update-nav`** - ナビゲーション更新
   - 新しいプロジェクトをサイトのナビゲーションに追加

#### 🚀 **基本的なワークフロー**

```bash
# 1. 完全なドキュメント生成
doc-full my-tool

# 2. 日本語翻訳
translate-docs my-tool

# 3. ナビゲーション更新
update-nav
```

詳細なテンプレートシステムについては、[テンプレートガイド](./templates.md)をご覧ください。

## 🔧 ツール作成後のドキュメント化ワークフロー

### 推奨手順

1. **ツール開発完了後**
   - Claude Codeでツールを完成させる
   - ツールのディレクトリが準備されている状態

2. **ドキュメント自動生成**
   ```bash
   # 完全なドキュメント生成（英語）
   doc-full [project-path|project-name]
   
   # 日本語翻訳
   translate-docs [project-name]
   
   # ナビゲーション更新
   update-nav
   ```

3. **ローカル確認**
   ```bash
   # 開発サーバー起動
   npm run docs:dev
   ```

4. **公開準備**
   ```bash
   # プロダクションビルド
   npm run docs:build
   ```

### ドキュメント管理のメリット

- **🎯 統一品質**: 標準化されたドキュメント構造
- **🔍 発見性向上**: 統一されたナビゲーション
- **🌐 多言語対応**: 日本語・英語での情報提供
- **⚡ 効率的**: 自動生成による時間短縮

## 📖 ドキュメンテーション標準

すべてのツールは以下の**5セクション構造**で統一：

1. **概要** - ツールの目的と主要機能
2. **インストール** - セットアップ手順
3. **使用ガイド** - 基本的な使い方とワークフロー
4. **APIリファレンス** - 完全なコマンド仕様
5. **トラブルシューティング** - よくある問題と解決法

詳細は `DOCUMENTATION_STANDARDS.md` を参照してください。

## 🏗️ サイト管理

### 本番公開

```bash
# プロダクションビルド
npm run docs:build

# ローカルプレビュー
npm run docs:preview
```

### GitHub Pages自動デプロイ

このリポジトリはGitHub Actionsによる自動デプロイに対応しています。`main`ブランチへのプッシュで自動的にサイトが更新されます。
