# VitePress Documentation System

Welcome to the VitePress-based documentation management system.

## Getting Started

Check out our [Getting Started Guide](/en/guide/getting-started) to learn how to use this system.

## System Features

### 🤖 Claude Code Integration
Automated documentation generation with 5 custom commands
- `doc-full` - Complete documentation generation
- `doc-api` - API reference generation
- `doc-update` - Update existing documentation
- `translate-docs` - Japanese translation
- `update-nav` - Navigation updates

### 📚 Unified Structure
Consistent 5-section structure for all tools
1. Overview
2. Installation
3. Usage
4. API Reference
5. Troubleshooting

### 🌐 Multi-language Support
Complete support for Japanese and English

### ⚡ Automated Workflow
Simple 3-step process: `doc-full` → `translate-docs` → `update-nav`
