# VitePress Documentation System

VitePressベースのドキュメント管理システムへようこそ。

## はじめに

このシステムの使い方については [はじめにガイド](/guide/getting-started) をご覧ください。

## システムの特徴

### 🤖 Claude Code統合
5つのカスタムコマンドによる自動ドキュメント生成機能
- `doc-full` - 完全なドキュメント生成
- `doc-api` - API リファレンス生成
- `doc-update` - 既存ドキュメント更新
- `translate-docs` - 日本語翻訳
- `update-nav` - ナビゲーション更新

### 📚 統一された構造
すべてのツールで一貫した5セクション構造
1. 概要 (Overview)
2. インストール (Installation)
3. 使用方法 (Usage)
4. API リファレンス (API Reference)
5. トラブルシューティング (Troubleshooting)

### 🌐 多言語対応
日本語・英語での完全サポート

### ⚡ 自動化ワークフロー
`doc-full` → `translate-docs` → `update-nav` の簡単3ステップ

## ツールの追加

新しいツールを追加する方法については [テンプレートガイド](/guide/templates) をご覧ください。

- 🚀 **高速サイト**: VitePressによる最適化されたパフォーマンス
- 🔧 **カスタマイズ可能**: 用途に応じて設定変更可能
